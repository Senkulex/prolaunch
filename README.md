# Profession Launch

Landing page for the "Profession Launch" service. This project helps Ukrainian and Russian-speaking expats in Belgium start legal freelance work via umbrella companies (SMart/Tentoo) without the complexity of opening a sole proprietorship (Zelfstandige).

## Goal
To provide a streamlined, high-conversion path for specialists to return to their professions legally and efficiently.

## Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Deployed on: [Netlify](https://prolaunch.netlify.app/)

## Key Features
- Dynamic multi-language support (UA/EN/RU).
- Multi-step screening quiz with history navigation.
- Responsive mobile-first design.
- SEO-optimized metadata.
